Bypass para Acceso Zajuna
=========================

Este complemento de Firefox permite el acceso a Zajuna,una plataforma del SENA, que actualmente está generando un error y que bloquea el acceso a los recursos de la plataforma.

Solo se debe instalar el complemento y reiniciar el navegador. No es necesario realizar ninguna configuración adicional.
